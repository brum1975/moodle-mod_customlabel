Traduction des libellés

En activant cette option, ce libellé est traductible. Vous devez entrer la clef de chaîne de traduction dans la valeur de l'item. Attention : cette possibilité n'est possible qu'en ajoutant les traductions de champs dans les fichiers de langue du module customlabel. Elle n'est donc accessible qu'à un développeur/intégrateur.
