Etiquettes personnalisées

Classe d'étiquette

La classe d'étiquette définit à la fois la structure des données qui peuvent être saisies dans l'étiquette personnalisée, et le rendu de sortie de ces données à travers un template de réalisation.

Les classes sont des "plugins" définis dans le répertoire "type" du répertoire du module. Il existe quelques classes prédéfinies :

  • Texte : Met en place un paragraphe de texte libre. Il peut constituer une alternative à l'étiquette simple, mais son but est surtout de pouvoir définir un "styling" particulier pour le texte, à travers son template et sa feuille de style locale.
  • Note pédagogique : Un paragraphe de texte simple, mais dont le "styling" peut être rendu spécifique pour cet usage. Cette classe se différenciera du texte simple essentiellement par le réglage d'accès des rôles. Sa configuration réservera son contenu aux enseignants éditeurs et non éditeurs.
  • Note de conception : Un paragraphe de texte simple, mais dont le "styling" peut être rendu spécifique pour cet usage. Cette classe se différenciera du texte simple essentiellement par le réglage d'accès des rôles. Sa configuration réservera son contenu aux enseignants éditeurs.
  • Note d'expertise : Un paragraphe de texte simple destiné à collecter des avis d'expert ou de conseil pédagogique sur un contenu de cours. Cette classe se différenciera du texte simple essentiellement par le réglage d'accès des rôles. Sa configuration réservera son contenu aux enseignants éditeurs et aux ingénieurs pédagogiques.
  • Boîte de commentaires : Un paragraphe de texte simple, mais dont le "styling" peut être rendu spécifique pour une sémantique de "commentaire dans le texte". Cette classe se différenciera du texte simple essentiellement par le réglage d'accès des rôles.
  • Métadonnées de cours : Cette classe est un exemple de marquage de métadonnées pour qualifier un cours Moodle. Elle utilise la possibilité de raccorder certains de ses champs de données à des tables préexistantes dans Moodle.
  • Métadonnées "auteur" : Cette classe est un exemple de marquage de métadonnées pour qualifier les auteurs d'un contenu pédagogique.

Les autres classes trouvées dans ce menu sont des classes définies localement par votre administrateur/intégrateur de plate-forme. Il est possible également que certaines classes ci-dessus aient été supprimées ou désactivées.
